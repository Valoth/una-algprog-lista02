# Desafios de Design de Lógica 

Este repositório contém a resolução de exercícios práticos de lógica de programação, focados na criação de fluxogramas para sistemas do cotidiano. O objetivo desta atividade foi exercitar o raciocínio lógico e a estruturação de processos decisórios.

## Exercícios Resolvidos

### 1. Sensor de Estacionamento 
Fluxograma que simula o comportamento de um sensor de ré, variando entre bipes contínuos, intermitentes ou ausência de som com base na distância detectada.

### 2. Filtro do Instagram 
Lógica de fluxo para aplicação de filtros e postagem de fotos, utilizando decisões em sequência para determinar se a imagem deve ser convertida para P&B e qual o destino final do arquivo (servidor ou galeria).

### 3. Caixa Eletrônico 
Algoritmo de validação de saque que verifica a disponibilidade de saldo antes de processar a transação, garantindo a integridade da operação financeira.

## Tecnologias e Ferramentas
* **Lógica de Programação** (Condicionais e Fluxogramas)
* **Ferramenta de Design:** Miro
* **Git & GitHub** para versionamento
